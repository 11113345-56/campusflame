# CampusFlame 🔥
Uganda's #1 campus dating platform connecting university students.

## Live Site
https://brilliant-cascaron-6e8289.netlify.app

## Tech Stack
- Frontend: HTML5, CSS3, Vanilla JavaScript
- Backend: Supabase (PostgreSQL, Realtime, Edge Functions, Storage)
- Payments: Pesapal API (MTN MoMo & Airtel Money)
- Hosting: Netlify

## Setup
Clone the repo, create `js/config.js` with your Supabase credentials and `js/pesapal.js` with your Pesapal keys. See Supabase docs for setup.
